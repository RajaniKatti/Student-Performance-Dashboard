# 📊 Student Performance Dashboard (Power BI)

## 📌 Overview

This project is an interactive Power BI dashboard designed to analyze and visualize student performance data. It provides insights into academic trends, subject-wise performance, and overall student progress.

---

## 🎯 Objectives

* Analyze student performance across subjects
* Identify high and low performing students
* Track academic trends and patterns
* Support data-driven decision making

---

## 📂 Files Included

* `Student Performance Dashboard.pbix` → Main Power BI dashboard
* `images/dashboard.png` → Dashboard preview

## 📊 Key Features

* Interactive filters and slicers
* Subject-wise performance analysis
* Student comparison metrics
* Visual KPIs for quick insights
* Clean and user-friendly design
